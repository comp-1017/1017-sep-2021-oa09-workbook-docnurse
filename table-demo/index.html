Table Demo

A table is a way of creating a structureed grid display of data. The paired tags (open/close) we will be interested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the header row of the table
  • <tbody> : identifies the data detail body of the table
  • <tfoot> : identifies the footer row of the table
  • <tr> : identifies the start of the row of the table
  • <th> : identifies the cell on the header row for the column
  • <td> : identifies the cell on the detail row or footer row to table

Some tags have parameters. The border parameter of the table tag applies a border around your table. The colspan parameter and rowspan parameter are use on the <td> tab to allow you to merger cloumns on a single row or column cels across rows.
